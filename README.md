## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS

## 💻 Projeto

Este projeto é parte de um dos desafios do curso Explorer da [Rocketseat](https://www.rocketseat.com.br/).

Nele fiz a recriação do layout do Figma com o intuito de trabalhar os itens abaixo:

- Estruturando um projeto HTML;
- Uso de tags HTML mais comuns;
- Conceitos iniciais de CSS;
- Posicionamento e alinhamento de elementos na tela;
- Aplicando fontes customizadas;
- Espaçamentos.


Além do que foi solicitado, adicionei mais algumas features que acho interessante como:

- Responsividade à página
- Animação no cateado inferior esquerdo da tela
